Getting Started

What is StegoCrypt Suite?

StegoCrypt Suite is a powerful desktop application that combines a Flutter-based frontend with a Python backend CLI to enable secure data hiding through:

  • Image Steganography using LSB encoding techniques
  • Audio Steganography with echo hiding methods
  • Video Steganography through frame-based embedding
  • Text Steganography using zero-width characters
  • AES Encryption with 128-bit keys
  • RSA Encryption for secure key exchange
  • Secure Hashing with multiple algorithm support
  • File Security with X25519 KEM + AES-GCM

System Architecture

StegoCrypt Suite follows a client-side architecture where a Flutter frontend communicates with a Python backend via direct process execution:

  • Frontend: Flutter UI (Dart) - provides the user interface and handles user interactions
  • Backend: Python CLI - performs all steganography, cryptography, and file security operations
  • Communication: Direct process execution - Flutter invokes the Python CLI as a subprocess
  • Data Exchange: JSON responses - all backend responses are formatted as JSON for easy parsing

Quick Start

  1. Download StegoCrypt Suite for your platform
  2. Install the application (requires Python 3.8+ and FFmpeg)
  3. Launch and create your first project
  4. Select your steganography method
  5. Choose encryption parameters
  6. Process your files securely

Installation Guide

System Requirements

Minimum Requirements:

  • Processor: Intel Core i3 or equivalent
  • RAM: 4GB (8GB recommended)
  • Storage: 500MB available space
  • Graphics: Basic graphics card
  • Python: Python 3.8 or later
  • FFmpeg: FFmpeg installed and available on system PATH

Supported Operating Systems:

  • Windows 10 or later
  • macOS 10.14 or later
  • Ubuntu 18.04 or equivalent Linux distributions

Required Python Dependencies

The Python backend requires the following dependencies to be installed:

  • Pillow==10.1.0 - Image processing library
  • pycryptodome==3.19.0 - Cryptographic toolkit
  • cryptography>=41.0.0 - Cryptographic recipes and primitives
  • pydub - Audio manipulation library
  • numpy - Numerical computing library
  • opencv-python - Computer vision library for video processing
  • ffmpeg-python - FFmpeg wrapper (optional but recommended)

Installation Steps

Windows:

  1. Download StegoCrypt_Suite_v3.0.0_Windows.exe
  2. Install Python 3.8+ if not already installed
  3. Install FFmpeg and ensure it's available on your system PATH
  4. Run the installer as administrator
  5. Follow the setup wizard
  6. Launch from Start Menu or Desktop shortcut

macOS:

  1. Download StegoCrypt_Suite_v3.0.0_macOS.dmg
  2. Install Python 3.8+ if not already installed
  3. Install FFmpeg and ensure it's available on your system PATH
  4. Open the DMG file and drag to Applications
  5. Launch from Launchpad or Applications

Linux:

  1. Download StegoCrypt_Suite_v3.0.0_Linux.AppImage
  2. Install Python 3.8+ if not already installed
  3. Install FFmpeg and ensure it's available on your system PATH
  4. Make executable: chmod +x StegoCrypt_Suite_v3.0.0_Linux.AppImage
  5. Double-click to run or execute from terminal
  6. Optionally integrate with desktop environment

X25519 + AES-GCM

Advanced file encryption using modern cryptographic techniques for maximum security.

  • AES-256-GCM - Authenticated encryption with integrity verification

  • Metadata inspection for encrypted files

System Architecture

Overview

StegoCrypt Suite implements a client-side architecture with a clear separation between the user interface and the computational backend:

Frontend (Flutter)

  • Built with Flutter framework for cross-platform compatibility
  • Provides intuitive UI for all steganography and cryptography operations
  • Handles user input validation and error presentation
  • Manages application state and user preferences

Backend (Python CLI)

  • Single-process CLI entrypoint: stegocrypt_cli.py
  • Executed via direct process execution from Flutter
  • Returns all responses as JSON for easy parsing
  • Implements all core steganography and cryptography algorithms

Communication Protocol

  • Flutter invokes Python CLI using Process.run()
  • Commands follow pattern: python stegocrypt_cli.py <command> [--args]
  • All responses are JSON objects printed to stdout
  • Error handling through standardized JSON error responses

Module Structure

  • Steganography: image_stego.py, audio_stego.py, video_stego.py, text_stego.py
  • Cryptography: aes_crypto.py, rsa_crypto.py, hashing.py
  • File Security: file encryption and secret sharing utilities
  • Utilities: validation, logging, and helper modules
